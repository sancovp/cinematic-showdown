# Quest: cinematic_site_001 — First Cinematic Site

## Objective

Build your first cinematic website from scratch. Choose any URL, capture it, design it cinematically, and deploy it live.

## Requirements

- Step 1: `capture.json` — Valid JSON with url, title, sections, assets, fonts, color_palette, veo_prompt
- Step 2: `DESIGN.md` — Design system with brand name, colors, fonts, motion language
- Step 3: `copy.md` — Hero headline, subheadline, at least 2 section headlines, CTAs
- Step 4: `index.html` — Single-file HTML with GSAP, responsive, liquid glass, grain
- Step 5: Deployed live URL — GitHub Pages URL accessible via curl
- Step 6: QC report — Pass at least 10 of 15 quality gate items

1. Page loads without JS errors
2. Hero section visible above fold
3. Veo video plays (or placeholder)
4. Instrument Serif font loads
5. Liquid glass on at least one card
6. Grain texture visible
7. Dark theme applied
8. rAF crossfade on scroll
9. Responsive at 1440px
10. Responsive at 768px
11. Responsive at 375px
12. CTAs functional
13. No layout shift on load
14. Lighthouse Performance ≥ 70
15. Lighthouse Accessibility ≥ 70

Pass 10/15 to complete the quest.
