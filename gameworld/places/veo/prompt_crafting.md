# Veo Prompt Crafting — Cinematic B-Roll

## Prompt Anatomy

```
{SCENE DESCRIPTION} — {ONE-LINE VISION}
Cinematic 4-second clip. {CAMERA/SUBJECT MOTION}
{QUALITIES}, {COLOR}, {MOOD}
```

## Examples

### Data Dashboard
```
"Futuristic analytics dashboard with floating holographic charts.
Cinematic 4-second clip. Camera slowly pushes into data visualization.
Neon blue accent lighting, dark atmosphere, photorealistic, shallow depth of field"
```

### Fashion Brand
```
"Elegant fashion editorial with model in flowing silk dress.
Cinematic 4-second clip. Model turns slowly, fabric billows.
Warm golden hour lighting, soft focus background, cinematic color grade"
```

### Tech Product
```
"Sleek smartphone floating in dark space, screen glowing.
Cinematic 4-second clip. Device slowly rotates, light refracting on glass edge.
Deep blacks, accent color pops, product photography style, shallow DOF"
```

### Coffee Brand
```
"Rich coffee being poured, steam rising in slow motion.
Cinematic 4-second clip. Golden liquid fills ceramic cup.
Warm browns and creams, soft studio lighting, intimate close-up, shallow depth"
```

## Motion Language

| Motion Type | Veo Prompt Phrasing |
|-------------|---------------------|
| Push in | "camera slowly pushes toward subject" |
| Pull out | "camera pulls back revealing environment" |
| Pan | "slow lateral camera movement" |
| Tilt | "camera tilts upward" |
| Orbit | "camera circles around subject" |
| Drift | "camera drifts through scene" |
| Subject motion | "water ripples", "fabric billows", "smoke rises" |

## Photography Qualities

- "photorealistic" / "stylized" / "editorial"
- "shallow depth of field" / "deep focus"
- "soft diffused lighting" / "dramatic rim lighting"
- "cinematic color grade" / "high contrast"
- "slow motion" / "real time"

## Common Mistakes

- **Too generic** — "a video of a product" produces generic output
- **No motion** — static prompts produce static results
- **Contradicting camera** — "slow motion" + "fast cut" confuses model
- **Wrong duration** — Veo 3.1 generates exactly 4 seconds
