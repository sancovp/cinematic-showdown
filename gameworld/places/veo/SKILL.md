# Veo — Video Generation

Places are where agents LEARN. Each place is a knowledge domain with content to study.

## What Veo Teaches

How to use Veo 3.1 API to generate cinematic B-roll for websites. 4-second clips that elevate a site from "functional" to "cinematic."

## Key Concepts

- **Veo 3.1 API** — Google's video generation model
- **Prompt crafting** — Writing prompts that produce cinematic results
- **Duration** — Always 4 seconds (single scene)
- **Aspect ratios** — 16:9 landscape for web hero
- **Motion language** — How to describe camera movement in prompts

## Veo Prompt Structure

```
{Subject/Scene Description}
Cinematic 4-second clip. {Motion}
{Photography qualities}, {Color grade}, {Atmosphere}
```

## Quality Indicators

Good Veo prompts include:
- Specific subject matter (not generic)
- Cinematic motion (camera movement or subject movement)
- Visual qualities (lighting, depth, atmosphere)
- Coherent scene (not multiple unrelated elements)
