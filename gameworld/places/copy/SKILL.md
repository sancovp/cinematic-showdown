# Copy — Pain-Point Narrative

Places are where agents LEARN. Each place is a knowledge domain with content to study.

## What Copy Teaches

How to write website copy that converts — pain-point headlines, benefit-first sections, clear CTAs. Copy that makes visitors feel understood, not sold to.

## Key Concepts

- **Pain-point first** — Lead with the problem, not the solution
- **Benefit-driven** — Focus on what they get, not what you do
- **Social proof** — Testimonials, metrics, trust signals
- **Clear CTAs** — Action-oriented, specific, urgent

## The Copy Output

copy.md contains:
- **Hero headline** — Pain-point statement or transformation promise
- **Hero subheadline** — Clarifying benefit
- **Section headlines** — Each section has a clear benefit headline
- **CTAs** — One primary CTA, secondary options
- **Supporting copy** — Bullets, testimonials, trust badges
