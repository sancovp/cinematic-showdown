# HTML — Single-File Cinematic Pages

Places are where agents LEARN. Each place is a knowledge domain with content to study.

## What HTML Teaches

How to build a complete, cinematic single-file HTML page. GSAP animations, responsive breakpoints, liquid glass effects, film grain — everything in one self-contained file.

## Key Concepts

- **Single-file architecture** — All CSS, JS inline, no external dependencies except CDN
- **GSAP** — Animation library for scroll-triggered effects
- **Responsive** — 1440px, 768px, 375px breakpoints
- **Performance** — Lazy loading, efficient selectors
- **Accessibility** — Proper heading hierarchy, contrast, focus states
